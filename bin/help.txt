=== Usage: ===
   On Linux use: ./gol-designer
   On Windows use: gol-designer

   Available commands:
      portfolio install - Installs dependencies needed to build
      portfolio clean - Cleans build directory
      portfolio build - Builds development version (run gol-designer install before building otherwise errors will appear!)
      portfolio release - Build release version
      portfolio server - Starts autobuilds server at localhost:3000
